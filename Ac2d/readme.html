Ac2d - 2D Acoustic Finite-Difference simulation of seismic waves

This directory contains the source files for the Ac2d simulation library. The library is written in a smal Domain Specific Language (DSL) and converted to either standard c, CUDA or c with OpenMP pragma's using a source-to-source translator (transpiler). The transpiler is named eps and found in a separate github repo: Eps

List of source files

  • mk.sh : Script for compiling the source code
  • ac2d.e : Solver object
  • diff.e : Differentiator object
  • model.e : Model object
  • rec.e : Receiver object
  • src.e : Source object
  • ac2dmod.e : Example code to run the library

Tests

The Test directory contains simple tests and plotting routines for initial verification.
